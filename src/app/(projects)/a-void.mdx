Game fore Ludum Dare 50, solo-developed in 48h.

Controls:

Movement: WASD
Throw/Recover light: LMB
Move light while throwing: Hold LMB + Mouse movement
Stop recovering: RMB
Restart: R
The game doesn't have an End State. Have fun!



The source code: guillemsrr/AVoid_LudumDare50

The retro shader I used: DaveFace/UnrealRetroShaders: Retro PS1 Post Processing for Unreal Engine 4
