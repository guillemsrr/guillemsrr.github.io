Game solo-developed in one day for the GMTK Game Jam 2019, theme: "Only One".

Defend your green cube from the enemies.
You can only use one laser at time, change it with the button next to the green cube.

Right now the game is "unwinnable" it will keep getting harder, adding a cube for each level won, and having to defend a bigger space.
