It's using [SerraEngine](https://github.com/guillemsrr/SerraEngine) my custom simple game engine that I use
to improve my skills in low-level programming.

Following the book [RayTracing in one weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html)
as well as The Cherno's [YouTube ray tracing series](https://youtu.be/gfW1Fhd9u9Q?si=FUIoSHL7A7SPaMHo) (superb, highly recommended!).

[GitHub source (WIP)](https://github.com/guillemsrr/raytracing-audio)
